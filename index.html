AI Behavior

This AI behavior demo contain some real time strategy mechanics. Map scrolling, tile grid rendering and custom bounds collision. Bots are generated by CPU entities. These bot enteties find and gather resources. They are designed to find and harvest the closes resource and move on once depleted. I also added some mutation logic that would allow some bot to become self aware and eventualy break free from the parent CPU's control. The CPU entities can be selected and then toggle behaviors on it's menu. Defence mode summon all bot's to stop harvesting and return to the CPU's area.
